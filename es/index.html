Proyectos

HistoryMap Aplicación Web (2022 - actualidad)

HistoryMap es mi proyecto personal, desarrollado con React y Firebase, que permite a los usuarios crear mapas históricos personalizados con marcadores detallados de eventos significativos. Diseñada para educadores de historia, esta plataforma ofrece una herramienta valiosa para mostrar eventos históricos contextualizados en un mapa interactivo del mundo. Con experiencia en desarrollo front-end y experiencia en el uso de Firebase, gestioné la implementación de características clave, incluida la gestión de permisos para mapas privados y funcionalidades próximas, como el intercambio de mapas y un sistema de notificaciones en curso. La integración de LeafletJS proporciona una experiencia de mapa fluida y atractiva. El sitio web es receptivo y está diseñado cuidadosamente con Tailwind CSS.
